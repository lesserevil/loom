# Changelog

All notable changes to the AgentiCorp VS Code extension will be documented in this file.

## [1.0.0] - 2026-01-21

### Added
- Initial release
- AI chat panel in sidebar
- Context menu actions for selected code:
  - Ask about selection
  - Explain code
  - Generate tests
  - Refactor code
  - Fix bugs
- Configuration options for API endpoint and preferences
- Auto-context injection for code-aware responses
- Conversation history management
- Error handling and user-friendly messages
- TypeScript implementation with full type safety

### Features
- Real-time chat with AgentiCorp
- Code selection context injection
- Syntax highlighting in chat
- Keyboard shortcuts
- Configuration UI

### Documentation
- Complete README with examples
- Configuration guide
- Troubleshooting section
- Development setup instructions
